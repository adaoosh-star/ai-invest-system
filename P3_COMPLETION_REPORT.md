## 二、功能详情

### 2.1 MD&A 深度解读

**分析方法**:
| 方法 | 功能 | 状态 |
|------|------|------|
| 规则式分析 | 情感倾向、关键词统计 | ✅ |
| LLM 增强分析 | 管理层语气、可信度、关键成就、风险因素 | ⚠️ (API 超时时降级) |

**规则式分析指标**:
- 积极词汇提及次数
- 消极词汇提及次数
- 风险提及次数
- 情感倾向（积极/中性/消极）
- 情感评分（-1 到 +1）

**LLM 增强分析指标**:
- 关键成就（3-5 个）
- 主要挑战（2-3 个）
- 未来展望（积极/中性/消极）
- 战略重点（2-3 个）
- 风险因素（3-5 个）
- 管理层语气（自信/谨慎/乐观/悲观）
- 可信度评分（0-1）

**测试结果**:
```

  摘要：情感倾向：积极 (0.25)
  情感倾向：积极
```

### 2.2 跨年度趋势分析

**分析指标**:
- 营收（revenue）
- 净利润（net_profit）
- 毛利率（gross_margin）
- ROE（roe）
- 现金流比率（cash_flow_ratio）
- 负债率（debt_ratio）
- 研发费用率（rd_expense_ratio）

**趋势计算**:
- CAGR（复合年增长率）
- 波动率（标准差）
- 趋势方向（快速增长/稳定增长/小幅下滑/快速下滑）
- 整体趋势评估（整体向好/分化/整体承压/平稳）

**测试结果**:
```
跨年度趋势分析:
  分析年数：3
  整体趋势：整体向好
```

**示例数据**:

|------|------|--------|-----|--------|
| 2023 | 30 亿 | 5 亿 | 15% | 40% |
| 2024 | 38 亿 | 6.5 亿 | 18% | 42% |
| 2025 | 50 亿 | 8 亿 | 22% | 45% |

## 五、使用指南

### 5.1 测试年报分析模块

```bash
python3 run.py annual-test
```

### 5.2 在代码中使用

```python
from analysis.annual_report_analyzer import get_annual_analyzer

analyzer = get_annual_analyzer()

# MD&A 分析
result = analyzer.analyze_mda(mda_text, year=2025)
print(f"情感倾向：{result['rule_based']['sentiment']}")

# 跨年度趋势分析
trend = analyzer.analyze_trend(historical_data)
print(f"整体趋势：{trend['overall_trend']}")

# 生成完整报告
report = analyzer.generate_full_report(
    ts_code='002270.SZ',
    stock_name='华明装备',
    mda_text=mda_text,
    historical_data=historical_data
)
```
