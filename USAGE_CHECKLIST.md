# AI 价值投资系统 - 使用检查清单

**创建日期：** 2026-03-26  
**目的：** 确保每次股票分析都使用完整系统，禁止临时写脚本

---

## ✅ 分析前检查

### 1. 确认使用系统
- [ ] 我是否准备使用 `ai-invest-system/run.py`？
- [ ] 我是否避免了自己写临时脚本？
- [ ] 我是否记得 2026-03-26 嘉美包装的教训？

### 2. 确认命令格式
```bash
# 完整分析（推荐）
python3 run.py analyze <股票代码>

# 快速分析（仅 20 项检查）
python3 run.py analyze <股票代码> --quick
```

### 3. 确认股票代码格式
- [ ] 代码格式：`002969.SZ` 或 `600519.SH`
- [ ] 深交所股票：`.SZ` 后缀
- [ ] 上交所股票：`.SH` 后缀

---

## ✅ 分析后检查

### 1. 确认报告生成
- [ ] 报告是否保存至 `cache/` 文件夹？
- [ ] 报告文件名格式：`<代码>_完整报告_<时间戳>.md`

### 2. 确认分析内容
- [ ] 20 项检查清单是否完整执行？
- [ ] A 股风险检查是否包含？
- [ ] NLP 事实一致性校验是否执行？

### 3. 确认结论格式
根据《投资宪法》：
- **≥15 项✅ (75%)**：可买入
- **10-14 项⚠️ (50-74%)**：待查
- **<10 项❌ (<50%)**：放弃
