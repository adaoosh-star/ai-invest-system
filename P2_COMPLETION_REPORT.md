## 二、决策规则量化

### 2.1 买入决策（量化评分）

**评分权重**:
| 指标 | 权重 | 说明 |
|------|------|------|
| 20 项检查通过率 | 30% | 投资宪法执行度 |
| ROE 评分 | 25% | 资本回报率 |
| 现金流评分 | 20% | 盈利质量 |
| 估值评分 | 15% | PE 分位（越低越好） |
| 增长评分 | 10% | 营收增长率 |

**决策阈值**:
| 评分 | 决策 | 等级 | 建议仓位 |
|------|------|------|---------|
| ≥0.85 | 买入 | 强烈推荐 | 25% |
| ≥0.70 | 买入 | 推荐买入 | 20% |
| ≥0.50 | 观望 | 观望 | 10% |
| <0.50 | 避免 | 避免 | 0% |

**测试结果（华明装备）**:
```

评分：0.89
等级：强烈推荐
原因：20 项检查通过率 85%, ROE 连续 5 年>22%, 现金流/净利润 1.16
建议仓位：25%
```

## 五、使用指南

### 5.1 测试决策引擎

```bash
python3 run.py decision-test
```

### 5.2 在代码中使用

```python
from model.decision_engine_v2 import get_decision_engine

engine = get_decision_engine()

# 买入决策
result = engine.make_buy_decision(
    ts_code='002270.SZ',
    stock_name='华明装备',
    financial_data={...},
    checklist_result={'pass_rate': 0.85},
    current_pe_percentile=0.30,
    ...
)
print(f"决策：{result['decision']}")
print(f"评分：{result['score']}")
```

### 5.3 配置调整

编辑 `config/investment_constitution_v2.yaml`:
```yaml
scoring:
  thresholds:
    strong_buy: 0.85  # 调整阈值
    buy: 0.70
```
