## 🔧 配置步骤（2 分钟）

### 步骤 1：创建钉钉机器人

1. 打开钉钉 → 找到任意群（或创建一个新群）
2. 点击右上角 **⚙️ 设置**
3. 选择 **群机器人**
4. 点击 **添加机器人**
5. 选择 **"自定义"**（通过 Webhook 接入）

### 步骤 2：配置机器人

- **机器人名称：** AI 价值投资系统
- **Webhook 地址：** 复制并保存（重要！）
- **安全设置：** 选择 **"自定义关键词"**
  - 添加关键词：`AI 价值投资`
  - 添加关键词：`持仓监控`
  - 添加关键词：`盘前`
  - 添加关键词：`盘后`

### 步骤 3：保存 Webhook

Webhook 格式：
```
https://oapi.dingtalk.com/robot/send?access_token=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### 步骤 4：告诉我 Webhook

将 Webhook 地址发给我，我立即配置推送！

## 🔒 安全说明

- Webhook 只用于推送消息，无法读取您的钉钉数据
- 机器人只能发送到配置的群
- 建议创建一个专门的"投资报告"群
