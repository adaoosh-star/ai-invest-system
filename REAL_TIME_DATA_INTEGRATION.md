### 3. 盘后复盘报告（postmarket_report.py）

#### 接入数据
- **日线数据** - 开盘/最高/最低/收盘/成交量
- **均线** - MA5/MA10/MA20（自主计算）
- **MACD** - DIF/DEA/HIST（自主计算）
- **KDJ** - K/D/J（自主计算）
- **RSI** - 相对强弱指标（自主计算）

#### 数据源
- **Tushare Pro** - 日线数据、财务数据
- **numpy** - 技术指标计算

#### 技术指标算法

**均线（MA）：**
```python
MA5 = mean(close[-5:])
MA10 = mean(close[-10:])
MA20 = mean(close[-20:])
```

**MACD：**
```python
EMA12 = EMA(close, 12)
EMA26 = EMA(close, 26)
DIF = EMA12 - EMA26
DEA = EMA(DIF, 9)
MACD = (DIF - DEA) * 2
```

**KDJ：**
```python
RSV = (close - lowest_low) / (highest_high - lowest_low) * 100
K = SMA(RSV, 3)
D = SMA(K, 3)
J = 3*K - 2*D
```

**RSI：**
```python
delta = diff(close)
gain = where(delta > 0, delta, 0)
loss = where(delta < 0, -delta, 0)
avg_gain = mean(gain[-14:])
avg_loss = mean(loss[-14:])
RS = avg_gain / avg_loss
RSI = 100 - (100 / (1 + RS))
```

## 🛡️ 错误处理

### 降级策略

| 场景 | 降级方案 |
|------|---------|
| **隔夜市场获取失败** | 显示"数据获取失败"，不影响持仓分析 |
| **集合竞价获取失败** | 显示"数据获取失败"，使用成本价作为参考 |
| **日线数据获取失败** | 显示"数据获取失败"，跳过技术指标计算 |
| **技术指标计算失败** | 显示"-"，不影响价格数据展示 |

### 代码示例
```python
# 增强错误处理
try:
    data = fetch_data()
    if data and data.get('price'):
        # 处理数据
    else:
        # 降级处理
except Exception as e:
    logger.debug(f"获取数据失败：{e}")
    return {'current': 0, 'change': 0, 'comment': '数据获取失败'}
```
