## 🔧 修改内容

### 1. Cron 调度时间

**修改前：**
```json
{
  "schedule": {
    "kind": "every",
    "everyMs": 300000  // 5 分钟
  }
}
```

**修改后：**
```json
{
  "schedule": {
    "kind": "cron",
    "expr": "0,30 9-15 * * 1-5",  // 周一到周五 9:00-15:30 每 30 分钟
    "tz": "Asia/Shanghai"
  }
}
```

**含义：**
- `0,30` - 每小时的 0 分和 30 分
- `9-15` - 9 点到 15 点（覆盖交易时间 9:30-15:00）
- `* * 1-5` - 周一到周五

**执行时间点：**
```
09:00, 09:30, 10:00, 10:30, 11:00, 11:30, 12:00, 12:30,
13:00, 13:30, 14:00, 14:30, 15:00, 15:30
```

共 14 次/天（交易日）

### 2. 推送内容优化

**修改前：** 每次都输出详细报告

**修改后：**
```
根据预警级别输出：
- 🔴🟠🟢 有预警：输出详细报告摘要
- ✅ 正常：输出一行摘要
```

**推送格式：**

**有预警（详细）：**
```
🔴【持仓监控】10:30

华明装备 (002270) 补仓机会
- 现价：¥26.00（实时）
- 补仓位：¥26.00
- 建议：补仓 2000 股

详细报告：cache/holding_report_20260326_103003.md
```

**无预警（摘要）：**
```
【持仓监控】10:30 ✅ 正常 | 总市值¥315,720 (-11.78%) | 华明¥27.50 电网¥1.884
```

## 🎯 设计理念

### 1. 只监控有意义的时段

| 时段 | 监控有意义吗 | 原因 |
|------|-------------|------|
| 交易时间 | ✅ 有 | 可以补仓/减仓 |
| 盘后/盘前 | ❌ 无 | 无法交易 |
| 周末/节假日 | ❌ 无 | 市场关闭 |

### 2. 监控≠心跳

- **持仓监控** - 投资机会提醒（交易时间）
- **系统心跳** - 后台日志记录（不需要推送）

### 3. 简洁优先

- 无预警时一行摘要
- 有预警时详细报告
- 不为了"心跳"而增加推送

## 📝 用户须知

### 你会收到什么推送？

| 场景 | 推送内容 | 频率 |
|------|----------|------|
| **交易时间有预警** | 详细报告（补仓/估值等） | 发生时 |
| **交易时间无预警** | 一行摘要（正常） | 每 30 分钟 |
| **非交易时间** | ❌ 不推送 | - |

### 推送量

- **交易日：** 约 8-14 条/天
- **非交易日：** 0 条/天
- **平均：** 约 8 条/天（相比之前 288 条，减少 97%）

### 如何确认系统运行？

1. **交易时间** - 每 30 分钟收到"正常"摘要
2. **需要时** - 随时询问"系统运行正常吗？"
3. **后台日志** - 自动记录，不需要推送
